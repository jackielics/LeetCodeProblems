# Topic
Python3
# Intuition
Backtrack

# Approach
从空列表开始，依次遍历数组每个位置的元素，并开始新的回溯，在现有subset每个位置插入下一个元素；

# Complexity
- Time complexity:
$$O(n)$$

- Space complexity:
$$O(n!)$$

# Code
```python
class Solution:
	def permute(self, nums: List[int]) -> List[List[int]]:
		res = []

		def backtrack(cur:list):
			if len(cur) == len(nums):
				res.append(cur)
				return 
			nextElem = nums[len(cur)]

			for i in range(len(cur) + 1):
				backtrack(cur[:i] + [nextElem] + cur[i:])

		backtrack([])
		return res
```
