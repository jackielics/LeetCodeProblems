# Intuition
Hashset or slow fast pointer

# Approach
Hashset or slow fast pointer

# Complexity
- Time complexity:
$$O(n)$$

- Space complexity:
$$O(n)$$

# Code
```python
class Solution:
    def hasCycle0(self, head: Optional[ListNode]) -> bool:
        hashSet = set()
        p = head
        while p:
            if p in hashSet:
                return True
            else:
                hashSet.add(p)
            p = p.next
        return False

    def hasCycle(self, head: Optional[ListNode]) -> bool:
        if not head:
            return False

        slow = fast = head
        while slow:
            slow = slow.next
            fast = fast.next
            if not fast or not fast.next:
                return False
            fast = fast.next
            if slow == fast:
                return True
        return True
```
