# Intuition
Backtrack

# Approach
similar to https://leetcode.cn/problems/subsets/ ; sort and skip just popped elem

# Complexity
- Time complexity:
$$O(n)$$

- Space complexity:
$$O(n)$$

# Code
```python
class Solution:
    def subsetsWithDup(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        res = []
        subset = []
        def backtrack(i):
            if i >= len(nums):
                res.append(subset.copy())
                return

            subset.append(nums[i])
            backtrack(i + 1)

            popped = subset.pop()
            while i < len(nums) - 1 and nums[i + 1] == nums[i]:
                i += 1
            backtrack(i + 1)
        backtrack(0)
        return res
```
