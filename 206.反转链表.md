# Topic
206. Reverse Linked List

# Intuition
Linked List or Recursion

# Approach
Linked List or Recursion


# Complexity
- Time complexity:
$$O(n)$$

- Space complexity:
$$O(n)$$

# Code
```python
class Solution:
	def reverseList0(self, head: Optional[ListNode]) -> Optional[ListNode]:
		'''
		My Solution0: iterate using 3 pointers [prev, cur, next]
		Time Complexity: O(n)
		Space Complexity: O(n)
		'''
		pre, cur = None, head
		while cur:
			nxt = cur.next
			cur.next = pre
			pre, cur = cur, nxt
		return pre

	def reverseList1(self, head: Optional[ListNode]) -> Optional[ListNode]:
		# Recursion: Nested function
		def recurse(pre, cur):
			if not cur:
				return pre
			nxt = cur.next
			cur.next = pre
			return recurse(cur, nxt)
		return recurse(None, head)
```
